How modules work

  • Each module lives in modules/Name/ with a Module.php entry point
  • Controllers go in modules/Name/controllers/, models in modules/Name/models/
  • Disabled modules are skipped on boot — their classes and routes are not loaded
  • Disabled state is saved to storage/modules.json
  • Changes take effect on the next request (no restart needed)
